grafica_serie
=============

Graficador de series utilizado para ejemplos de algoritmos para acceso a disco en SO II - UCSE - DAR

Es una aplicación Bottle que pide una serie y un conjunto de parámetros para realizar el gráfico de la serie.

Está inicialmente pensada para graficar los accesos a disco según una lista de bloques, donde el orden de la lista depende de un algoritmo de acceso específico.

El parámetro de tiempo/potencia es para graficar los valores de tiempo utilizados en lugar de los bloques (tiempo de desplazamiento * nro de bloques).
